# Arc Template: Modular Panel Specification Framework
## Reusable Blueprint for Any Story Arc (6-8 Panels)

---

## INTRODUCTION: ARC-BASED STORYTELLING

This template allows you to create new story arcs indefinitely while maintaining:
- ✅ Visual consistency with established character designs
- ✅ Bitcoin narrative coherence
- ✅ Tier progression logic
- ✅ Enemy/challenge escalation
- ✅ Shareholder army growth

Each arc is **self-contained yet interconnected**, allowing the story to expand as real-world events unfold.

---

## PART 1: ARC ANATOMY (Structure Every Arc Follows)

### Standard Arc Structure: 6-8 Panels

```
Panel 1: Inciting Incident / Challenge Emerges
Panel 2: Initial Struggle / First Engagement
Panel 3: Deepening Conflict / Enemy Reveals True Nature
Panel 4: Low Point / Moment of Doubt
Panel 5: Breakthrough / New Understanding
Panel 6: Rising Action / Power Manifestation
Panel 7: [Optional] Preparation / Army Gathering
Panel 8: [Optional] Climax / Challenge Overcome
```

### Arc Emotional Arc

**Always follows this pattern:**
1. **Setup** (panels 1-2) - New challenge introduced
2. **Complication** (panels 3-4) - Obstacle seems insurmountable
3. **Crisis** (panel 4-5) - Lowest point, temporary despair
4. **Resolution** (panels 6-8) - Saylor's growth overcomes obstacle

### Arc Thematic Focus

Each arc should focus on **one aspect of Bitcoin thesis**:
- Arc 1: "Bitcoin is infinite duration"
- Arc 2: "Capital machine is executable"
- Arc 3: "Institutional resistance is surmountable"
- Arc 4: "Global adoption is inevitable"

## PART 3: INTER-ARC CONSISTENCY RULES

### Rule 1: Character Recognition
**Saylor's face must be recognizable across ALL arcs**
- Maintain 60% facial feature consistency
- Eyes evolve but same basic shape
- Scar on left temple always visible/glowing appropriately
- Hair color evolution happens slowly (black → dark purple → white-silver)

### Rule 2: Tier Progression Logic
**Tiers progress but don't skip**
- Arc 1: Tier 1 → Tier 1-2
- Arc 2: Tier 2 → Tier 2-3
- Arc 3: Tier 3 → Tier 3-4
- Arc 4+: Tier 4 → Tier 4+ (beyond human, cosmic scale)

### Rule 3: Enemy Escalation
**New arcs introduce new challenges**
- Arc 1: Skeptic Collective (weakness = ignorance)
- Arc 2: Internal system challenges (Premium Paradox, Limitation Leviathan)
- Arc 3: Institutional enforcement (S&P Mecha, Blindness Curse)
- Arc 4+: Global scale threats (adapt to real events)

### Rule 4: Summon Growth
**Summons become more visible and organized**
- Arc 1: Invisible
- Arc 2: Shadows → Clear individuals
- Arc 3: Hundreds visible, organized formations
- Arc 4+: Thousands, extending to horizon

### Rule 5: Color Language Consistency
**Primary colors for each character never change**
- Saylor: Gold-purple blend evolving toward white-gold
- STRK: Always red-gold
- STRF: Always bright gold
- STRD: Always silver-blue
- Skeptics: Always gray-red
- Institutional: Always gray-black

### Rule 6: Timeline Coherence
**If adding new arcs, they must fit real Bitcoin timeline**
- Don't skip major events
- Integrate real Bitcoin milestones
- Adapt to actual Saylor/MicroStrategy news
- Show how fictional story mirrors reality

## PART 5: CREATING NEW ARCS (Future Expansion)

### When to Create New Arc?

**Triggers for new arc creation:**
1. **Major Bitcoin Event** (ETF approval, price milestone, adoption)
2. **Major MicroStrategy Event** (new capital raise, partnership, acquisition)
3. **Major Institutional Event** (rating change, reserve recognition)
4. **Timeline Reaches New Phase** (next major quarter/year)
5. **Story Reaches Natural Stopping Point** (satisfying climax, sets up next chapter)

### How to Create New Arc

**Step 1: Define the Thesis**
- What Bitcoin truth is revealed this arc?
- What new understanding does Saylor gain?
- Example: "Arc 4 thesis: Bitcoin adoption is macro-scale inevitable"

**Step 2: Identify Real-World Timeline**
- When do these panels take place?
- What real events align?
- Example: "Arc 4 happens in 2026, Bitcoin ETF growth phase"

**Step 3: Design Enemy/Challenge**
- What's the logical next obstacle?
- Previous obstacles: Doubt, Internal paradox, Institutional resistance
- New obstacle: Macro economic forces? Governmental pressure? Competing narratives?

**Step 4: Determine Saylor's Tier**
- What tier does he START this arc in?
- What tier does he END in?
- Example: "Arc 4 starts Tier 4, evolves to Tier 4+ (cosmic scale)"

**Step 5: Plan Summon Evolution**
- If summons were visible in Arc 3, are they still visible?
- Do they evolve (new forms, more powerful)?
- Do new summons appear?

**Step 6: Map Shareholder Growth**
- How many shareholders visible?
- Which tranche (STRK/STRF/STRD/common)?
- Are they already organized, or organizing?

**Step 7: Fill In Using Template**
- Use Arc Template above
- Create 6-8 panels following the structure
- Verify consistency with all previous arcs
